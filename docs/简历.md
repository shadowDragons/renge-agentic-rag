**技术栈**：`Python` `FastAPI` `SQLAlchemy` `Pydantic Settings` `Alembic` `LangGraph` `LlamaIndex` `Qdrant` `SQLite` `PostgreSQL` `LangChain Core` `Vue 3` `TypeScript` `Vite` `Element Plus` `Pinia` `SSE` `Docker` `Docker Compose`

**项目简介**：负责搭建企业内部知识库 RAG 系统，围绕文档上传入库 -> 多知识库检索 -> 引用式问答 -> 人工审核恢复的核心链路，建设助理管理、知识库管理、文档解析入库、会话问答、任务中心、审核台、审计日志等模块，为 ERP 系统配套企业制度、业务资料、操作手册和高风险问题治理能力。

**工作内容**：

1. 基于 FastAPI + SQLAlchemy 设计后端服务结构，完成助理、知识库、文档、会话、审核、任务、审计日志等核心业务模块与 API 实现。
2. 基于 Vue 3 + TypeScript + Vite + Element Plus 开发管理后台，落地知识库管理、文档上传、聊天问答、审核台、任务中心、系统总览等页面与交互流程。
3. 基于 LlamaIndex + Qdrant 实现文档切块、向量入库、知识库范围过滤、dense retrieval + lexical rerank 检索链路，并支持引用片段回传。
4. 基于 LangGraph 搭建澄清、检索、审核挂起、审核恢复、答案生成工作流，支持高风险问题命中审核规则后中断，并在审核通过后恢复执行。
5. 设计并实现 SSE 流式问答、任务状态追踪、审计日志记录等能力，提升问答链路可观测性与系统可运维性。
6. 接入 Langfuse，完成 chat turn trace、retrieval / rerank / generation span、LLM usage / cost、人工审核 score、离线评测与 Prompt Management，支持回答质量评估、Prompt 版本回溯和线上问题定位。
