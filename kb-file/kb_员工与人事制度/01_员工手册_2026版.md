# 员工手册（2026版）

## 1. 适用范围
本手册适用于公司全体正式员工、试用期员工及经审批纳入统一管理的外包驻场人员。

## 2. 工作时间
- 标准工时：周一至周五 09:30 - 18:30
- 弹性打卡：上班可在 09:00 - 10:00 间完成打卡
- 午休时间：12:00 - 13:30

## 3. 远程办公规范
1. 需提前在协同系统提交申请。
2. 连续远程办公超过 3 天需直属主管审批。
3. 涉及客户数据处理时必须接入公司 VPN。

## 4. 保密义务
员工不得将未公开经营数据、客户名单、源代码、合同草案通过个人邮箱、网盘、即时通信工具私自外发。

## 5. 奖惩机制
- 季度绩效 A：可参与年度特别激励池。
- 严重违反信息安全制度：视情节给予警告、记过或解除劳动合同。
